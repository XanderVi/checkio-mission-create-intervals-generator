Из {% if interpreter.slug == "js-node" %}массива(Array) целых чисел(Integer){% else %}множества(set) целых чисел(int){% endif %} вам нужно создать список(list) замкнутых интервалов в виде {% if interpreter.slug == "js-node" %}массивов(Arrays){% else %}кортежей(tuple){% endif %} таких, чтобы интервалы охватывали все значения, найденные в множестве.
В этой миссии вам необходимо использовать 'yield', чтобы сделать вашу функцию генератором.

Замкнутый интервал включает в себя конечные точки! Интервал 1..5 , например, включает каждое значение x , которое удовлетворяет условию: 1 <= x <= 5.

Значения могут быть в одном интервале только если разность между значением и следующим меньшим значением в наборе равно единице, иначе начинается новый интервал.
Отдельное значение, которое не вписывается в существующие правила формирования интервалов, становится начальной и конечной точкой нового интервала.

Входящие данные: {% if interpreter.slug == "js-node" %}массив(Array) целых чисел(Integer){% else %}множество(set) целых чисел(int){% endif %}.

Исходящие: {% if interpreter.slug == "js-node" %}масссив массивов двух целых чисел(Array of Array of two Integers){% else %}список кортежей двух целых чисел(A list of tuples of two ints){% endif %}, обозначающими концы промежутка. Массив должен быть отсортирован по начальной точке каждого интервала.
